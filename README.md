## Getting Started

### Prerequisites

- Node.js 16.8 or later
- npm or yarn

### Installation

1. Clone the repository:


2. Install dependencies:
    ```npm install```

3. Start the development server:
    ```npm run dev```

4. Open your browser and navigate to `http://localhost:3000` to see the application in action.
